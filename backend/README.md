# Description

Backend for Store Admin. It uses NestJS (Express) located in `/backend`.
It stores products in `.json` file that has some mock data on server startup.
[Swagger located here](http://localhost:3000/api)

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
